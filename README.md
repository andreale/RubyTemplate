# About

A simple template for Ruby projects.

## Dependencies

* Ruby 3.4.2
* Bundler gem

## How to get started

* `rvm install ruby 3.4.2 --install --create`
* `gem install bundler`

* `bundle`

## How to run the test suite:

* `rspec`

## How to run the program

* `make`

## Run rubocop

* `rubocop`
